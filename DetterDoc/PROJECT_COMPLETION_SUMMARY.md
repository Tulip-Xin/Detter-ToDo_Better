## 📊 项目概览

Detter (ToDo Better) 是一款基于 React Native 开发的 Android 待办事项管理应用，核心理念是"一件事情要么紧急，要么重要"，并在任务完成后引导用户进行反思复盘。

### 核心特性

- ✅ 按优先级（重要/紧急/琐事）管理任务
- ✅ 完成任务后即时复盘
- ✅ 卡片列表动态缩放效果
- ✅ 数据统计和可视化
- ✅ 主题切换（浅色/深色）
- ✅ 数据导入导出
- ✅ 完全离线工作

## 🛠️ 技术实现

### 技术栈

```
React Native 0.72+
├── TypeScript 4.9+
├── React Navigation 6.x
├── React Native Reanimated 3.x
├── React Native Gesture Handler 2.x
├── SQLite Storage 6.x
├── Notifee 7.x
└── React Native Chart Kit 6.x
```

### 架构模式

- **UI 层**: 组件化设计，可复用组件
- **业务逻辑层**: Context + useReducer 状态管理
- **数据访问层**: Repository 模式，Service 层
- **存储层**: SQLite 本地数据库

- ✅ 冷启动时间: < 2 秒

1. **动态缩放效果**: 使用 Reanimated 实现流畅的卡片缩放

3. **性能优化**: 列表虚拟化、动画优化、数据库索引
4. **状态管理**: Context + useReducer 模式，清晰的数据流
