## [1.0.0] - 2025-11-18

### 🎉 首次发布

Detter (ToDo Better) v1.0.0 正式发布！这是一款基于 React Native 开发的 Android 待办事项管理应用，核心理念是"一件事情要么紧急，要么重要"，并在任务完成后引导用户进行反思复盘。

### ✨ 新增功能

#### 任务管理（行）
- 📝 按优先级（重要/紧急/琐事）管理待办事项
- 📅 横向滚动日期选择器，支持查看不同日期的任务
- 🎯 三个优先级容器，清晰展示任务分类
- ✅ 点击复选框标记任务完成，支持完成动画
- 📋 子任务功能，将大任务拆分成小步骤
- 🏷️ 标签系统，使用 # 快速添加和管理标签
- ⏰ 任务提醒，设置提醒时间避免遗忘
- 🗓️ 日历视图，显示公历和农历日期
- ✏️ 任务编辑，支持修改所有任务信息
- 🔄 拖拽排序，长按拖拽调整任务顺序
- 👆 左滑操作，快速归档、删除、上移、下移任务
- 🔽 下拉刷新，刷新当前日期的任务列表

#### 复盘反思（思）
- 🤔 即时复盘，完成任务后立即记录反思
- 📇 卡片列表视图，沉浸式浏览体验
  - 动态缩放效果，中心卡片最大
  - 自动居中对齐，停止滚动后自动吸附
  - 惯性滑动控制，最多滚动 3 张卡片
- 📄 清单列表视图，简洁列表展示
  - 按完成日期降序排序
  - 已复盘任务紫色背景标记
- 💭 复盘详情，查看和编辑复盘笔记
- 🔍 搜索功能，搜索任务标题、描述、标签、复盘内容
- 🎛️ 筛选功能，按日期范围、标签、复盘状态筛选

#### 个人中心（我）
- 📊 数据统计
  - 完成率统计（每日/每周/每月）
  - 任务分布图表（重要/紧急/琐事）
  - 复盘习惯统计（复盘率）
- 🎨 主题切换
  - 浅色模式
  - 深色模式
  - 跟随系统
- 🔔 通知设置，开启/关闭任务提醒
- 💾 数据导出，支持 JSON 和 CSV 格式
- 📥 数据导入，从文件导入任务和复盘
- 🗑️ 数据清空，清除所有数据（需确认）
- ℹ️ 关于和帮助，查看应用信息和使用帮助

### 🛠️ 技术特性

- ⚡ 高性能动画，使用 React Native Reanimated
- 🎯 手势支持，使用 React Native Gesture Handler
- 💾 本地存储，使用 SQLite 数据库
- 🎨 主题系统，支持浅色/深色主题
- 📱 响应式设计，适配不同屏幕尺寸
- 🔒 数据安全，所有数据存储在本地
- 🚀 快速启动，冷启动时间 < 2 秒
- 🎭 流畅动画，60fps 滚动性能

### 📱 平台支持

- Android 9.0 (API 28) 及以上版本
- 支持 armeabi-v7a, arm64-v8a, x86, x86_64 架构

## 版本说明

### 版本号格式

版本号格式为 `主版本号.次版本号.修订号`，例如 `1.0.0`

- **主版本号**: 重大功能更新或不兼容的 API 变更
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 变更类型

- **新增 (Added)**: 新功能
- **变更 (Changed)**: 现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug 修复
- **安全 (Security)**: 安全相关的修复
