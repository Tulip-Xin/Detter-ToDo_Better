## 使用说明

### 用户文档使用

1. **新用户**:
   - 阅读 `README.md` 了解项目
   - 查看 `docs/USER_GUIDE.md` 学习使用
   - 参考常见问题解决疑问

2. **遇到问题**:
   - 查看 `docs/KNOWN_ISSUES.md`
   - 查找对应的解决方案
   - 联系开发者反馈

### 开发者文档使用

1. **新开发者**:
   - 阅读 `docs/DEVELOPER_GUIDE.md`
   - 了解架构和技术栈
   - 按照环境搭建步骤配置

2. **开发过程**:
   - 参考开发规范编码
   - 查看核心模块说明
   - 使用测试指南编写测试

3. **准备发布**:
   - 查看 `docs/ACCEPTANCE_TEST_CHECKLIST.md`
   - 参考 `docs/RELEASE_PREPARATION.md`
   - 更新 `CHANGELOG.md`

### 测试文档使用

1. **功能测试**:
   - 打开 `docs/ACCEPTANCE_TEST_CHECKLIST.md`
   - 逐项检查所有功能
   - 记录测试结果

2. **发布前验证**:
   - 完成所有测试项
   - 确认通过率
   - 记录问题和建议

### 发布文档使用

1. **准备阶段**:
   - 阅读 `docs/RELEASE_PREPARATION.md`
   - 准备所有资源
   - 更新版本信息

2. **打包阶段**:
   - 按照指南构建 APK
   - 验证签名和功能
   - 准备发布材料

3. **发布阶段**:
   - 选择发布渠道
   - 上传应用和资源
   - 提交审核
